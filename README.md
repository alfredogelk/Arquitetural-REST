# Arquitetural-REST
Explorando o Estilo Arquitetural REST com Node.js- Conheça o estilo arquitetural mais utilizado na atualidade e entenda todas as nuances do REST. Nesse contexto, explore na prática todo potencial semântico do protocolo HTTP com Node.js. Para isso, o expert utiliza a biblioteca Express para abstrair o gerenciamento de rotas.
